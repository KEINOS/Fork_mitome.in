# mitome.in
Explore OpenPGP as an alternative for seals (mitome-in). View the site alive at https://mitome.in/

## Usage
```sh
yarn install
```

Run locally:

```sh
yarn docs:dev
```

Build the pages:

```sh
yarn docs:build
```

Static files are available under `./docs/.vuepress/dist/`.
